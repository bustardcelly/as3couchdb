documentproperty 
document:CouchDocument

Internal Accessor/Modifier for related document. This is internal and is only set in parsing JSON object returned from COuchDB. To add attachments to a CouchDocument instance, use the CouchDocument#attachments modifier. When a CouchDocument is saved, it will look to see if a correlating CouchDocument is associated with this CouchAttachment. If not it will push the document as well.
